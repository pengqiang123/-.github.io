什么是Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装

安装之前你必须先下载git以及Node.js。安装完毕后，打开Git Bash依次输入下列指令:

创建博客的本地文件

安装完毕后，我们可以开始搭建我们的博客啦，将它取名为myblog，输入下列指令：
$ hexo init myblog
$ cd myblog
$ npm install
这时，我们就可以开始编辑网页了。
